# DIGITAL LITERACY PROJECT

Name: Vaishnavi CH  
Register Number: 25BCY10117  
Course: CSE0001 Digital Literacy  
Institution: VIT BHOPAL

This Digital Literacy Project is part of the CSE0001 course. 
It includes five tasks such as infographic creation, digital portfolio, coding, email etiquette, and cybercrime awareness. 
The project helps students understand digital tools and use them safely and responsibly.

Task 1: Digital Literacy Infographic
Created an infographic using Canva to spread awareness about safe internet usage.

Task 2: Digital Portfolio
Created GitHub, LinkedIn, and Kaggle profiles and uploaded project files.

Task 3: Coding and Collaboration
Completed a HackerRank coding challenge and created a Google Form.

Task 4: Email Etiquette
Wrote professional emails and created social media Do’s and Don’ts.

Task 5: Cybercrime Awareness
Studied phishing and created a prevention checklist.
